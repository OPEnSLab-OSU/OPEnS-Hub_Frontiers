# OPEnS-Hub_Frontiers
This is the Data, CAD, and Code archive for the OPEnS Hub Project for access by Frontiers

[Project Page](http://www.open-sensing.org/lora-hub)

[Project Loom Repository](https://github.com/OPEnSLab-OSU/InternetOfAg) that this is a instance/subset of 

## Development Boards

This project used the Adafruit line of Development boards called "Feathers" and "Featherwings" 

Specifically 
* Adafruit's  M0 LoRa Feather
* Adafruit's Ethernet FeatherWing 
* Adafruit's Adalogger FeatherWing (Real-time Clock and SD Card)
